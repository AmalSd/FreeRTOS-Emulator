void vDrawUpdateScreen ( void  )

Executes the queued draw jobs.

The tumDraw functions are designed to be callable from any thread, as such each function queues a draw job into a queue. Once vDrawUpdateScreen is called, the queued draw jobs are executed by the background SDL thread.

Returns
NULL always return NULL
